# GROSHI.online

Responsive landing page for promoting the long-term lease of the **groshi.online** domain.

## About

This repository contains the source code for a single-page landing site prepared for **GitHub Pages** deployment.

The page presents the domain as a branding and marketing asset for businesses in the financial niche, including:

- online lending services
- microfinance platforms
- pawnshops
- finance-related lead generation projects

## Features

- responsive desktop and mobile layout
- sticky header with anchor navigation
- compact mobile burger menu
- highlighted contact button in the header
- back-to-top button
- smooth anchor scrolling
- sticky-header offset handling for section navigation
- lightweight structure suitable for static hosting

## Tech Stack

- HTML5
- CSS3
- vanilla JavaScript
